Usage
-----
bt pull [/merge parent] [<repo>]
  merge: Pull changes from parent repository into child worktree
  repo:  repository or worktree on which to perform the pull operation
         (if repo is not given, the current directory is used)
  Note:  (For svn repositories this is an "svn update")

Description
-----------
Pulls updates from an upstream location into a local repository or worktree.

See Also
--------
bt push
bt select

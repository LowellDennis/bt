Usage
-----
bios status [all]
     all:  Include status of unversioned files as well
     NOTE: All may be abbreviated (e.g. 'a' is the same as 'all')

Description
-----------
Shows the VCS status for the current BIOS worktree (or repository)
1. Change directory to top of BIOS worktree
2. Issue VCS status command
   (Filter out all but meaningful changes)
3. Change directory back to original directory

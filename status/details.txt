Description
-----------
Get the status of the current BIOS repository or worktree.

Usage
-----
bios status [all]
     all:  Include status of unversioned files as well
     NOTE: All may be abbreviated (e.g. 'a' is the same as 'all')

Steps
-----
1. Mark the starting directory
  a. In case of an error, change the directory to the starting directory.
2. Change directory to the top of the repository or worktree.
  a. Error: if the top cannot be found!
  b. Error: if the top is not in the list of repositories or worktrees!
3. Filter the output of the status command showing changes to any versioned files.
  a. If all is given, also show changes to unversioned files.
4. Change the directory to the starting directory.

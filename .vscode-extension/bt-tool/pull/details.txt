Description
-----------
Updates a local repository or worktree with changes from its remote location.

Usage
-----
bt pull [<path>]
  path:  repository or worktree on which to perform the pull operation
         (if path is not given, the current directory is used)

Steps
-----
1. Validate command line arguments.
  a. Error: if command line arguments are invalid!
2. The target directory is the path, if provided,
   or the current directory if not.
3. Determine the top directory of the target repository or worktree.
  a. Error: if the top cannot be found!
  b. Error: if the top is not in the list of repositories or worktrees!
  c. Error: if this repository or worktree has uncommitted changes.
4. Perform the appropriate operation,
  a. For git, pull.
  b. For SVN, update.

See also
--------
push, select, switch

Description
-----------
Add a BIOS repository to the list of repositories.

Usage
-----
bt attach [<path>]
   path: Path associated with the BIOS repository to be attached.
         (Defaults to current directory.)

Steps
-----
1. Validate command line arguments.
  a. Error: if command line arguments are invalid!
2. The target directory is the path, if provided,
   or the current directory if not.
3. Determine the top directory of the target repository or worktree.
  a. Error: if the top cannot be found!
  b. Error: if the top is not a repository!
  c. Error: if the top is already in the list of repositories!
4. Add the top directory to the list of repositories.
5. If no repository is selected, the newly attached repository is selected.

See also
--------
detach, pull, push, select, switch

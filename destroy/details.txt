Usage
-----
bt destroy /branch [<path>]
   branch: Delete the branch associated with path (if any)
   path:   Path associated with BIOS worktree to be deleted
           (defaults to current directory)

Description
-----------
This command destroys a BIOS worktree.

NOTE: The BIOS worktree will be deleted from the disk
      Ending directory will be parent of worktree base if defaults are used

See Also
--------
bt create
bt pull
bt push

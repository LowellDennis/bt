Description
-----------
Remove a BIOS repository from the list of repositories.

Usage
-----
bt detach [<path>]
   path:  Path associated with BIOS repository to be detached
          (defaults to current directory)

Steps
-----
1. Mark the starting directory.
  a. In case of an error, change the directory to the starting directory.
2. If a path is given, change the directory to the path.
3. Change directory to the top of the repository or worktree.
  a. Error: if the top cannot be found!
  b. Error: if the top is not a repository!
  c. Error: if the top is not in the list of repositories!
4. Remove the repository from the list of repositories.
5. Change back to the starting directory.

See also
--------
attach, pull, push, select, switch
